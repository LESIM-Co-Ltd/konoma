以下はすべて既定値で、config の `[keys.<面>]` で画面ごとに変更できます(→
[設定リファレンス](../configuration/))。アプリ内では `?` が
その画面のキーを表示します。

## グローバル(全画面共通)

| キー | 動作 |
|---|---|
| `?` | 現在の画面のヘルプ |
| `Q` | 終了(`ui.confirm_quit = false` でなければ確認。`qq` で素早く) |
| `t` | タブ新規(閉じるはツリーの `q`。`w` は vim の単語移動と誤爆するため既定未割当) |
| `[` / `]` / `1`-`9` | 前 / 次 / 番号のタブ |
| `F` | フォローモード(ディスク上の変更を自動表示) |
| `T` | タブ一覧(切替 / 閉じる。タブバーは幅超過時に `‹n / n›` で隠れタブ数を表示) |

## ツリー

| キー | 動作 |
|---|---|
| `j` `k` / `g` `G` | 移動 / 先頭 / 末尾 |
| `l` | ディレクトリへ潜る(ファイルはプレビュー) |
| `h` | 親ディレクトリへ |
| `Enter` | その場で展開/折りたたみ(ファイルはプレビュー) |
| `/` | 入力しながら絞り込み(`Esc` で解除) |
| `.` | 隠しファイル切替 |
| `s` / `i` / `r` | ソートメニュー / ファイル情報 / 更新 |
| `e` | 外部エディタで開く |
| `p` | パス表示の巡回(相対 / `~` / フル) |
| `v` / `V` | ビジュアル範囲選択 / 1件選択トグル |
| `Space` → `n r d c x p` | 作成 / リネーム / 削除 / コピー / カット / ペースト |
| `y` → `n r f p @` | 名前 / 相対 / フル / 親 / `@参照` をコピー |
| `m` + 英字 / `'` | ブックマーク登録 / 一覧 |
| `a` / `A` | 表示基準を現在地に / 起動位置に戻す |
| `o` / `d` / `O` | git 変更ハブ / カーソルファイルの diff / 外部 git ツール |
| `C` / `n` `N` | 変更ファイルビュー / 変更ファイル間ジャンプ |
| `q` | タブを閉じる(最後の1枚なら終了) |

## テキスト / Markdown プレビュー

| キー | 動作 |
|---|---|
| `j` `k`・`Ctrl-d` `Ctrl-u`・`g` `G` | スクロール / 半ページ / 先頭末尾 |
| `h` `l`・`0` `$` | 横スクロールと行頭/行末(折返しオフ時) |
| `/` `n` `N` | 検索 / 次 / 前の一致 |
| `Tab` / `Shift-Tab` | リンク/チェックボックスのフォーカス移動(md) |
| `Enter` | リンクを開く / チェックボックスをトグル |
| `Space` | フォーカス中チェックボックスをトグル |
| `R` | 装飾表示 ⇄ raw ソース(Markdown/Mermaid) |
| `v` / `V` | 文字単位 / 行単位の選択 → `y` でコピー |
| `Y` | `@path#L12-34` 参照をコピー |
| `Ctrl-n` / `Ctrl-p` | 次 / 前のファイルをプレビュー(ツリー表示順・端で wrap) |
| `m` + 英字 / `'` | 表示中ファイルをブックマーク / 一覧 |
| `e` / `q` | 外部エディタ / ツリーへ戻る |

## 画像 / PDF プレビュー

| キー | 動作 |
|---|---|
| `+` `-` / `0` `=` | ズーム / フィットに戻す |
| `h j k l` | パン |
| `J` `K` | PDF ページ送り / 戻し |
| `Ctrl-n` / `Ctrl-p` | 次 / 前のファイルをプレビュー(ツリー表示順・端で wrap) |
| `m` `'` / `e` / `q` | ブックマーク / エディタ / 戻る |

## CSV / TSV テーブルプレビュー

| キー | 動作 |
|---|---|
| `h j k l` | セル移動 |
| `Ctrl-n` / `Ctrl-p` | 次 / 前のファイルをプレビュー(ツリー表示順・端で wrap) |
| `g` `G` / `0` `$` | 先頭/末尾行 / 先頭/末尾列 |
| `y` → `c r C f` | セル / 行 / 列 / フルパスをコピー |
| `m` `'` / `e` / `q` | ブックマーク / エディタ / 戻る |

## Git

| キー | 動作 |
|---|---|
| ハブ(`o`): `s` `u` / `S` `U` | ファイルを stage/unstage / 全ファイル |
| ハブ: `c` / `x` / `Enter` | コミット / 変更破棄 / diff を開く |
| ハブ: `l` / `g` / `b` | log / コミットグラフ / ブランチ |
| diff: `s` / `n` `N` / `x` | レイアウト巡回 / 次/前の変更ファイル / 破棄 |
| log・グラフ: `Enter` / `y` | コミット詳細(全文+diff) / コミット情報コピー |
| グラフ: `s` / `x` / `b` | 基準ブランチ固定 / 解除 / ブランチ選択パネル |
| ブランチ: `Enter` / `n` / `d` / `/` | チェックアウト / 作成 / 削除 / 絞り込み |

## タブ一覧(`T`)

| キー | 動作 |
|---|---|
| `1`-`9` / `Enter` | そのタブ / 選択タブへ切替 |
| `j` `k` | 移動 |
| `d` | **選択タブ**を閉じる(一覧は開いたまま・最後の1枚は拒否) |
| `T` `q` `Esc` | 一覧を閉じる |

## ブックマーク一覧(`'`)

| キー | 動作 |
|---|---|
| 任意の英字 | そのブックマークへジャンプ(a-z ローカル / A-Z グローバル) |
| `j` `k` + `Enter` | 選択してジャンプ |
| `Ctrl-e` / `Ctrl-d` | 対象をエディタで開く / ブックマーク削除 |
| `'` `q` `Esc` | 閉じる |

一覧やグローバルのキーに使われている英字(`j` `k` `q` `t` `T` `F` `Q`)は
直接ジャンプできません — `j`/`k` + `Enter` で選ぶか、キーを再割当してください。
